# ONEMOREDOT. webpack starter pack

## 概要

LPつくるとき環境構築色々めんどくさいので、これ使うと楽よ。

以下の知識が必要よ。

- pug (これ読んどき。 https://tech.qookie.jp/posts/info-pug-feature/)
- scss
- babel(ES6)

## 必要なもの

- node v12.11.0 (nodenv管理が楽ちん)
- npm 6.11.3 (npxとかでつかうよ)

## インストール

`$ npm install`

## タスク

サーバー走らせながら開発するとき  
`$ npm run start:dev`
